# Battleship Game (C++)

A command-line implementation of the classic Battleship game, developed in C++.  
The project supports both **player vs player** and **player vs AI** gameplay modes, and demonstrates object-oriented design and core game system implementation.

---

## 🎮 Features

- Player vs Player mode  
- Player vs Computer (AI) mode  
- Turn-based gameplay system  
- Board management and ship placement validation  
- Hit / miss detection and win condition logic  
- Input validation and handling of edge cases  

---

## 🧠 Technical Highlights

- Designed using **object-oriented programming (OOP)** principles  
- Modular architecture separating game logic, board representation, and player behavior  
- Efficient handling of game state and turn progression  
- Clear separation between game rules and user interaction  

---

## 🏗️ System Design

The project is structured around key components such as:

- **Game Engine** – Controls game flow, turns, and win conditions  
- **Board System** – Manages grid representation and ship placement  
- **Player Module** – Handles player input and actions  
- **AI Logic** – Implements automated decision-making for the computer player  

---

## ⚙️ Technologies Used

- C++  
- Standard Template Library (STL)  
- Object-Oriented Programming
